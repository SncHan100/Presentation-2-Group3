AI can support medical interpreting by handling simple, low‑risk communication and assisting interpreters with tasks like terminology lookup or draft translations. However, it has major limitations—it cannot handle nuance, clarify meaning, or safely manage clinical conversations. Because of these risks, humans and institutions remain fully responsible for communication accuracy, and AI should only be used as a support tool, not a replacement.
